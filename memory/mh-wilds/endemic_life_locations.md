# Monster Hunter Wilds Endemic Life Locations

Last refreshed: 2026-05-07

This note covers non-fish endemic life. For aquatic life, fish, fishing bait, and Kanya fishing quests, use `fishing_locations.md`.

## Capture Rules

- Use the Capture Net from the item/radial bar. Aim until the reticle turns orange, then fire.
- Use Ghillie Mantle for skittish creatures. Sandstar can still detect you, so use Sonic/Screamer Pods or net it from maximum range.
- The detailed map can filter many endemic-life icons after discovery. Use it before manually sweeping.
- Captured endemic life can be checked/released through Ferdinand at Windward Plains base camp.
- There are 68 collectable non-aquatic endemic life entries in current guide data; some event/collab creatures may appear after updates.

## Practical Routes

- Samin chain route: Tracktail Lizard in Plains 11/13, Dapperwing in Forest 6/7/12/15/17, Ember Rufflizard in Basin 2/7/8/9, Rime Beetle near Cliffs 2, Gillopod in Wyveria 1/8/9/14, then Sandstar in Plains 13/16 at night.
- Scarlet Forest collection route: set Plenty daytime for Dapperwing, Dazzlewing, Jewel Scarab, Mimiphyta, Myriads, Vaporonid; rest to night for Hercudrome / Gold Hercudrome and Hymstrigian.
- Oilwell Basin route: Basin 13/14 during Plenty/Fallow covers Blink Angler, Pink Landlight Shrimp, Peony Sparklerjelly, Sparklerjelly, Gloom Gekko nearby if Fallow daytime.
- Ruins of Wyveria route: Plenty daytime for Gillopod, night for Prism Hercudrome, dusk/night for Xu Yo.
- Iceshard route: Fallow daytime/dusk for Downy Crake, then use map filter for Rime Beetle, Gawkie, Pillopod, Heatmantle Bug, and Solanaria.
