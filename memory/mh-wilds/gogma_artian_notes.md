# Gogma Artian Weapon Notes

Last refreshed: 2026-05-02

## Unlock / System

Gogma Artian weapons are upgraded Rarity 8 Artian weapons introduced with Title Update 4 / Gogmazios. They unlock after HR100 and defeating Gogmazios.

Upgrading a Rarity 8 Artian weapon into a Gogma Artian weapon:

- Requires Tarred Devices from Gogmazios.
- Keeps the existing Artian reinforcement bonuses.
- Adds a chosen focus: Attack, Affinity, or Element.
- Adds one random Group Skill and one random Set Bonus Skill.
- Opens Gogma Reinforce / amend options that can improve reinforcement tiers or reset them.

## Focus Choices

General focus effects from Mobalytics:

- Attack focus: +10 Attack, -15% Affinity.
- Affinity focus: +10% Affinity, -10 Attack, and an element adjustment.
- Element focus: +50 Element, -5% Affinity.

Game8 notes weapon-specific element adjustments. For Lance:

- Affinity focus: Lance element change is -20.
- Element focus: Lance element change is +50.

## Gogma Reinforcement Values

Gogma reinforcement tiers can go up to EX.

Common reinforcement values:

- Attack Boost I/II/III/EX: +5 / +6 / +9 or +8 depending source / +12.
- Affinity Boost I/II/III/EX: +5% / +6% / +8% / +10%.
- Sharpness Boost I/EX: +30 / +50.
- Lance Element Boost I/II/EX: +50 / +60 / +90.

Note: Mobalytics lists Attack III as +8, while Game8 lists +9. Verify in-game when optimizing exact math.

## Rerolling Skills

Gogma Artian weapons have rerollable random Group Skill and Set Bonus Skill.

Game8 reroll notes:

- You need two Rarity 8 Artian weapons of the same type and element for the save/reroll method.
- Tarred Devices with the same focus count for double points when resetting skills.
- The reroll process can be manipulated by testing on a spare matching weapon, returning to title without saving, then applying the known roll count to the main weapon.

## Skill Targets

Game8 broadly ranks these highly for Gogma Artian weapons:

- Group: Lord's Soul best; Lord's Fury recommended; Fortifying Pelt and Guardian's Pulse usable.
- Set bonuses: Gore Magala's Tyranny, Leviathan's Fury, Seregios's Tenacity, Jin Dahaad's Revolt, Fulgur Anjanath's Will, Ebony Odogaron's Power, Omega Resonance, Soul of the Dark Knight, Gogmapocalypse.
- Also recommended: Rey Dau's Voltage, Nu Udra's Mutiny, Guardian Arkveld's Vitality, Arkveld's Hunger, Zoh Shia's Pulse, Xu Wu's Vigor.

For the current Gore/Antivirus lance build, Gore Magala's Tyranny is especially valuable if it lets armor pieces be changed while preserving Black Eclipse / Antivirus synergy.

## Lance-Specific Heuristic

For a high-affinity Gore/Antivirus lance build:

- Affinity focus is usually lower value because Antivirus + Weakness Exploit can already push practical weak-point affinity near cap.
- Attack focus is attractive for general raw DPS if the -15% affinity does not push weak-point affinity too low.
- Element focus is attractive on element-matched hunts, especially with dragon lance into dragon-weak monsters; it only costs -5% affinity and adds +50 element for Lance.
- Reinforcement priorities are usually Attack, Element, Sharpness, then Affinity only if the build is not overcapping.
